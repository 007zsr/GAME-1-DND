# litter_box

这个目录用于存放“准备删除的文件”的删除前备份。

执行规则：
- 任何人只要准备在本项目里完全删除文件，必须先复制一份到 `respository root/litter_box/`，再执行删除。
- 复制到 `litter_box` 时，尽量保留原始相对路径，或至少在文件名中写明原路径与日期。
- 如果删除的是多个相关文件，建议在 `litter_box` 下按日期或任务名建子目录集中存放。
- 如果只是临时停用，不确定是否还会回退，优先不要直接删；至少先放一份备份到这里。

推荐命名：
- `litter_box/2026-04-10_player_menu_ui_v2.gd`
- `litter_box/2026-04-10_task_x/game/scripts/example.gd`

注意：
- 这里是“删除前备份箱”，不是正式资源目录。
- 这里的文件默认不参与现役主链路，只作为回退与追溯用。
