# GAME-1-DND

一个基于 Godot 4 的个人游戏项目仓库。

## 当前分支说明

- `main`：当前现役开发主线
- `archive/pre-2026-04-13-main`：线上旧版本归档分支

## 运行环境

- Godot 4.2.x

## 本地运行

1. 用 Godot 打开仓库根目录下的 `project.godot`
2. 直接运行项目

## 项目结构

- `character_creation/`：角色创建流程
- `game/`：主要 gameplay、场景、脚本、资源
- `main_menu/`：主菜单
- `tools/`：回归检查和开发辅助脚本

## 说明

- 公开仓库只保留正式工程内容、资源和必要说明
- 本地协作记录、阶段总结、内部整理文档不进入公开主线
